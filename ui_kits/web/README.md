# TourIstmo Web UI Kit

## Overview
High-fidelity React recreation of the TourIstmo web platform. Covers the core user flows with click-through navigation.

## Screens
1. **Homepage** — floating navbar, hero slider, destination & plan carousels, news section, footer
2. **Destinations List** — grid view with search + filters
3. **Destination Detail** — image, description, rating, map placeholder
4. **Login** — email/password form, social login
5. **User Profile** — saved destinations, tabs
6. **Admin Dashboard** — moderation queue with approve/reject

## Components
- `Navbar.jsx` — floating pill navbar with search
- `Footer.jsx` — teal footer with links
- `HeroSection.jsx` — full-bleed hero with slider controls
- `DestinationCard.jsx` — card with save/heart toggle
- `LoginForm.jsx` — auth form with social buttons
- `Dashboard.jsx` — admin panel with stats + table

## Usage
Open `index.html` in a browser. All navigation between screens is simulated client-side.
